# 가로등 배너 예약 서버

## 주요 기능

### 사용자 기능
1. 가로등 배너 구간 조회  
2. 가로등 배너 예약 현황 확인  
3. 가로등 배너 예약 신청  

### 관리자 기능
1. 구간 설정  
2. 회원 가입 관리  

## 기술 스택

- **서버**: Spring Boot, Spring Web MVC, Spring Data JPA, Spring Security  
- **데이터베이스**: MySQL
